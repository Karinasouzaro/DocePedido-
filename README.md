# DocePedido - Mockup Mobile

Este é um mockup premium de interface mobile para o app DocePedido, desenvolvido com HTML5, CSS3 e JavaScript vanilla.

## Funcionalidades

- **🏠 Home**: Saudação, barra de pesquisa, categorias, produtos em destaque e promoções.
- **🍰 Produto**: Detalhes do produto com imagem, descrição, preço, personalização e botão de encomenda.
- **📦 Pedidos**: Lista de pedidos ativos com status e prazo.
- **💰 Financeiro**: Resumo de vendas, lucro, gastos e gráfico simples.
- **👤 Perfil**: Informações da confeiteira, avaliações e configurações.

## Design

- Paleta: Rosa claro, branco, bege, nude, marrom suave.
- Estilo: Moderno, sofisticado, delicado, clean, elegante.
- Elementos: Cards modernos, ícones minimalistas, animações suaves, micro interações, navegação inferior fixa.

## Como visualizar

1. Abra o arquivo `index.html` em um navegador web.
2. Use as ferramentas de desenvolvedor (F12) para simular um dispositivo móvel.
3. Navegue pelas seções usando a barra inferior.
